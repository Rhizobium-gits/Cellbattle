## 🎮 ゲーム概要

**Cell Battle** は、ターン制カードバトルに、大学で学ぶ細胞生物学の知識を融合させた教育ゲームです。

<link>https://cell-battle.vercel.app</link>をホーム画面に追加して遊んでください。


プレイヤーは **インスリン** や **コラーゲン** 、**ヘモグロビン** といった実在のタンパク質をカードとして使い、エネルギーを消費して攻撃・結合（結合可能なものは同じ色で点滅）・サポートを駆使しながら相手と戦います。遊んでいるうちに、自然とタンパク質の種類・機能・分子量といった知識が身につきます。

### ✨ 特徴

- **122種類のタンパク質カード** — シグナル、構造、酵素、受容体、輸送、モーター、免疫など15カテゴリ
- **5種類のエネルギーシステム** — グルコース、ATP、Ca²⁺、電気信号、GTPを使い分ける本格メカニクス
- **結合（バインド）システム** — カード同士を結合させ、攻撃力を相乗的に上昇させる独自の戦略要素
- **サポートカード** — 回復・攻撃力倍化・バインド解除・蘇生など9種類の即時効果スペル
- **ドラッグ＆ドロップ操作** — カードを場にドラッグして直感的にプレイ
- **22種類のタンパク質形状SVGイラスト** — 球状、繊維状、Y字型など、実際の構造をモチーフにした美しいカードデザイン
- **★1〜★5のレアリティ** — 分子量と攻撃力に基づくレアリティシステム
- **AI対戦（3難易度）** — やさしい・ふつう・つよいの3段階で戦略が変化
- **アカウントシステム** — 称号、コイン、51種類のアバター
- **水彩画風の温かいデザイン** — 子どもにも親しみやすいアースカラーパレット

## 🃏 遊び方

### 基本ルール

1. **デッキ** — 各プレイヤーは100枚以上のデッキから5枚の手札でスタート
2. **勝利条件** — 相手のタンパク質を **3体倒す** と勝利
3. **ターン制** — 毎ターン、カードを1枚引き、エネルギーを1つ獲得

### ターンの流れ

| 順番 | アクション | 説明 |
|------|-----------|------|
| 1 | ドロー | デッキから1枚引く |
| 2 | エネルギー | 5種類からランダムに1つ獲得 |
| 3 | カードを出す | 手札からベンチにドラッグ（何枚でもOK） |
| 4 | 攻撃 or 結合 or サポート | 1ターンに1回だけ攻撃系アクションを実行 |
| 5 | ターン終了 | 「ターン終了」ボタンを押す |

### 攻撃

- アクティブカード（場の最前線）の攻撃ボタンを押す
- 攻撃にはエネルギーが必要（カードごとに必要なエネルギーの種類と数が異なる）
- ダメージは `攻撃力 + 結合ボーナス − 相手のシールド` で計算

### 結合（バインド）

- ベンチのカードをアクティブカードにドラッグすると **結合** できる
- 結合したカードは攻撃力にボーナスを与える（パートナーの攻撃力×0.25 + レアリティ差ボーナス）
- アクティブが倒されると、結合グループ全体が墓地へ（戦略的リスク）

### サポートカード

手札のサポートカード（⚡マーク）をフィールドにドラッグして即時効果を発動:

| 効果 | 説明 |
|------|------|
| ⚡ エネルギー | 指定エネルギーを即座に獲得 |
| 🃏 ドロー | カードを追加で引く |
| ⚔ 攻撃力UP | 今ターンの攻撃力を倍化 |
| 🗡 貫通 | シールドを無視してダメージ |
| 🛡 シールド | ダメージを軽減 |
| 💚 回復 | アクティブのHPを回復 |
| 🔓 バインド解除 | 結合を解除して手札に戻す |
| ♻ 蘇生 | 墓地からカードを手札に戻す |

## 🎯 操作方法

| 操作 | 方法 |
|------|------|
| カードを出す | 手札からベンチエリアへ **ドラッグ** |
| 結合する | ベンチのカードをアクティブカードへ **ドラッグ** |
| サポートを使う | サポートカードをフィールドへ **ドラッグ** |
| 攻撃する | アクティブカードの攻撃ボタンを **タップ** |
| 交代する | ベンチのカードを **タップ** |
| 結合解除 | 結合中のカードの 🔓 ボタンを **タップ** |
| カード詳細 | カードを **長押し** または **ダブルタップ** |
| ターン終了 | 「ターン終了」ボタンを **タップ** |

> 💡 スマートフォンでもPCでもそのまま遊べます。レスポンシブ対応済み。

## 🚀 セットアップ（ローカル実行）

Wi-Fi環境が不安定でも遊べるよう、**localhost:3000** で完全にオフライン動作します。

### 必要なもの

- [Node.js](https://nodejs.org/) v18以上
- npm（Node.jsに付属）

### 手順

```bash
# 1. リポジトリをクローン
git clone https://github.com/Rhizobium-gits/cell-battle.git

# 2. ディレクトリに移動
cd cell-battle

# 3. 依存関係をインストール
npm install

# 4. 開発サーバーを起動
npm run dev
```

**🌐 ブラウザで `http://localhost:3000` を開いてプレイ！**

> 💡 **同一ブラウザの2つのタブ** で開けば、ローカル対戦も可能です（localStorageを経由してタブ間で通信します）。

### スマートフォンで遊ぶ場合

同じWi-Fiに接続した状態で、PCのローカルIPアドレスを使ってアクセスできます:

```bash
# PCのIPアドレスを確認（例: 192.168.1.10）
# スマートフォンのブラウザで以下を開く:
# http://192.168.1.10:3000
```

## 🧑‍🔬 開発の背景

このゲームは、**子どもたちが細胞生物学に親しむきっかけ** として開発しました。

教科書の中のタンパク質は名前と機能の羅列になりがちですが、カードゲームとして手に取ることで「インスリンって結構強いんだ」「コラーゲンはHP高い！」「p53は転写因子なのか」といった発見が自然に生まれます。

すべてのカードの説明文は小学生にも理解できる言葉で書き直してあり、難しい専門用語は避けています。ゲームとして純粋に楽しみながら、気づいたら122種類のタンパク質を覚えている — それがこのゲームの目指すところです。
